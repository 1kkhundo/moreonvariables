function- a “chunk” of code that you can use over and over again, rather than writing it out multiple times.
function declaration-  tells the compiler about a function's name, return type, and parameters.
function call- an expression that passes control and arguments (if any) to a function
arguments- a value that is passed between programs, subroutines or functions.
type casting- When the conversion automatically performs by the compiler without the programmer's interference, it is called implicit type casting or widening casting.
